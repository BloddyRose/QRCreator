# QR Creator Tool
	Using the QRCoder library for .NET
	I a made a simple app that creates qr codes
 	

> Thanks QRCoder Contributors

>View the project here
[QRCodes](https://github.com/codebude/QRCoder)

## Features 
- 	 Text to QR code,
- 	 Wifi to QR Code,
- 	 Url to QR code
